# 线性代数

线性代数的基本问题是求解线性方程组

## Ax = b

给定两个方程：`2x - y = 0` 和 `-x + 2y = 3`。
对于`row picture` 矩阵是
```
[2, -1] [x] = [0]
[-1, 2] [y]   [3]
A        X  =  b
```
